# GUISO Project Roadmap

This document outlines the strategic phases for the evolution of the GUISO Token ecosystem.

## 📍 Phase 1: SocialFi MVP (Current)
*Focus: User Experience & Concept Validation*
- [x] Responsive Dashboard UI.
- [x] Simulated Token Economy (GSO Balance).
- [x] Impact Score System (IP).
- [x] Project Listing & Support Simulation.
- [x] Local Persistence (LocalStorage).

## 🔗 Phase 2: Web3 Integration
*Focus: Connecting to the Blockchain*
- [ ] **Wallet Connection:** Integration with MetaMask, Coinbase Wallet, and WalletConnect.
- [ ] **On-chain Data:** Reading real GSO balances from the blockchain.
- [ ] **Profile Verification:** Linking social profiles to wallet addresses.
- [ ] **Gasless Voting:** Implementation of EIP-712 for community signatures.

## 🧪 Phase 3: Blockchain Testnet
*Focus: Real Transactions*
- [ ] **Smart Contract Deployment:** GSO Treasury and Impact contracts on Testnet (Base/Polygon).
- [ ] **Real Support Actions:** Moving from simulated "Support" to real token transfers.
- [ ] **Automated Rewards:** Smart contracts distributing IP points automatically.

## 📸 Phase 4: Impact Proof System
*Focus: Verifiable Evidence*
- [ ] **Multimedia Evidence:** Linking photos/videos of social aid to specific transaction IDs.
- [ ] **Oracle Integration:** Using decentralized oracles to verify real-world project milestones.
- [ ] **Impact NFTs:** Minting soulbound tokens for high-impact contributors.

## 🏛️ Phase 5: Future DAO
*Focus: Full Decentralization*
- [ ] **Governance Portal:** Full on-chain voting for project selection.
- [ ] **Treasury Management:** Multi-sig community-managed funds.
- [ ] **Global Scaling:** Expanding the GUISO model to international social organizations.
