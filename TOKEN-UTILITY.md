# Utilidad del Token GSO y Economía Social

Este documento define la utilidad y el modelo económico del **Token GUISO (GSO)** dentro de la plataforma.

## 💎 Descripción del Token
- **Nombre:** GUISO Token
- **Símbolo:** GSO
- **Suministro Total:** 100,000,000
- **Red:** Binance Smart Chain (BSC)
- **Estándar:** BEP-20

## 🛠️ Utilidad Actual (MVP)
En la fase actual de MVP, la economía del token está **simulada** para demostrar el flujo del usuario.

### 1. Apoyo Social
Los usuarios pueden asignar GSO a proyectos sociales específicos.
- **Mecanismo:** Simulación de "Donación".
- **Beneficio para el Usuario:** Contribución directa a una causa.

### 2. Puntos de Impacto (IP)
Cada acción social genera **Puntos de Impacto**.
- **Fórmula:** `100 GSO Aportados = 10 IP`.
- **Utilidad:** Los IP determinan el `Nivel de Comunidad` del usuario.

### 3. Niveles de Comunidad
Los niveles representan la reputación del usuario dentro del ecosistema GUISO:
- **Aspirante:** 0 - 499 IP
- **Agente GUISO:** 500 - 1,999 IP
- **Guardián:** 2,000 - 4,999 IP
- **Embajador:** 5,000 - 9,999 IP
- **Leyenda:** 10,000+ IP

## 🚀 Tokenomics Futuros (V1)
Al pasar a la red principal (mainnet), se implementarán las siguientes utilidades:

### 1. Staking de Gobernanza
Los titulares pueden hacer staking de GSO para recibir **veGSO** (GSO de custodia de voto), lo que les permite:
- Proponer nuevos proyectos sociales.
- Votar sobre las asignaciones del tesoro.

### 2. Impuesto del Tesoro
Un pequeño porcentaje de cada transacción de GSO (ej. 1-2%) se envía automáticamente al **Tesoro Social de GUISO**.

### 3. Pruebas de Impacto Verificadas
GSO se utilizará para recompensar a los "Impact Scouts"—miembros de la comunidad que verifican físicamente el progreso de los proyectos sociales y suben evidencia.

### 4. Descuentos de Socios
Colaboraciones con marcas con conciencia social donde los titulares de GSO reciben beneficios exclusivos.

## 📊 Parámetros de la Economía Simulada
- **Balance Inicial del Usuario:** 10,000 GSO (para pruebas).
- **Ratio de Impacto:** 0.1 (10%).
- **Persistencia:** LocalStorage (Basado en sesión).
