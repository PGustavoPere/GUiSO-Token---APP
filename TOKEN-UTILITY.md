# GSO Token Utility & Social Economy

This document defines the utility and economic model of the **GUISO Token (GSO)** within the platform.

## 💎 Token Overview
- **Name:** GUISO Token
- **Symbol:** GSO
- **Total Supply:** 100,000,000
- **Network:** (Planned) BSC / Binance Smart Chain

## 🛠️ Current Utility (MVP)
In the current MVP phase, the token economy is **simulated** to demonstrate the user flow.

### 1. Social Support
Users can allocate GSO to specific social projects.
- **Mechanism:** "Donation" simulation.
- **User Benefit:** Direct contribution to a cause.

### 2. Impact Points (IP)
Every social action generates **Impact Points**.
- **Formula:** `100 GSO Supported = 10 IP`.
- **Utility:** IP determines the user's `Community Level`.

### 3. Community Levels
Levels represent the user's reputation within the GUISO ecosystem:
- **New Member:** 0-50 IP
- **Social Initiator:** 51-200 IP
- **Senior Collaborator:** 201-500 IP
- **Community Warrior:** 500+ IP

---

## 🚀 Future Tokenomics (V1)
When transitioning to the mainnet, the following utilities will be implemented:

### 1. Governance Staking
Holders can stake GSO to receive **veGSO** (voting escrow GSO), allowing them to:
- Propose new social projects.
- Vote on treasury allocations.

### 2. Treasury Tax
A small percentage of every GSO transaction (e.g., 1-2%) is automatically sent to the **GUISO Social Treasury**.

### 3. Verified Impact Proofs
GSO will be used to reward "Impact Scouts"—community members who physically verify the progress of social projects and upload evidence.

### 4. Partner Discounts
Collaborations with social-conscious brands where GSO holders receive exclusive benefits.

## 📊 Simulated Economy Parameters
- **Initial User Balance:** 10,000 GSO (for testing).
- **Impact Ratio:** 0.1 (10%).
- **Persistence:** LocalStorage (Session-based).
