# Modelo de Impacto: Traduciendo Tokens en Ayuda 🍲

El corazón de GUISO no es el código, sino cómo este código se traduce en un cambio positivo para las personas. El **Modelo de Impacto** es el conjunto de reglas que rigen esta transformación.

## 1. El Token GSO (Energía de Ayuda)
El token **GSO (BEP-20)** actúa como el vehículo de la ayuda. No es solo una moneda; es una unidad de compromiso social.
- **Uso:** Los usuarios aportan GSO a causas específicas.
- **Transparencia:** Cada movimiento de GSO es rastreable en la red BSC.

## 2. Puntos de Impacto (IP)
Para incentivar y reconocer la participación, el sistema genera **Puntos de Impacto (IP)** cada vez que un usuario colabora.
- **Cálculo:** Por cada 100 GSO aportados, el usuario recibe 10 IP.
- **Significado:** Los IP representan la "huella social" que el usuario está dejando en el mundo.

## 3. Niveles de Comunidad
Los IP acumulados determinan el nivel del usuario dentro del ecosistema. Estos niveles no son solo rangos, sino grados de responsabilidad y reconocimiento:

| Nivel | Rango de IP | Significado |
| :--- | :--- | :--- |
| **Aspirante** | 0 - 499 | Alguien que está dando sus primeros pasos en el ecosistema. |
| **Agente GUISO** | 500 - 1,999 | Un colaborador activo que ya ha generado un cambio tangible. |
| **Guardián** | 2,000 - 4,999 | Un pilar de la comunidad que vela por la transparencia. |
| **Embajador** | 5,000 - 9,999 | Un líder que inspira a otros a sumarse al propósito. |
| **Leyenda** | 10,000+ | Alguien cuyo impacto ha transformado vidas de forma histórica. |

## 4. El Bucle de Impacto
Este es el proceso circular que mantiene vivo al proyecto:
1. **Aporte:** El usuario entrega recursos (GSO).
2. **Acción:** El comercio o la ONG transforma esos recursos en ayuda (comida, kits).
3. **Evidencia:** Se genera un registro (on-chain) y una prueba visual del impacto.
4. **Reconocimiento:** El usuario sube de nivel y la comunidad crece en confianza.

## 5. Neutralidad y Transparencia
El modelo de impacto es **matemático y neutral**. No discrimina por ideologías. El único factor que determina el nivel y el reconocimiento es la **acción real y verificada**.
